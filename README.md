# 音乐管理项目开发文档

## DDD建模
### 统一语言
* 用户：指听音乐的人。
* 音乐信息：包括音乐标题、专辑名、歌手、文件位置。
* 歌单：每个用户有多个歌单，一个歌单有多个歌曲。
* 存储：一个本地文件目录，系统将会扫描这个文件夹，更新音乐库的信息。

### 聚合根
* 用户认证
    * ASP.NET Identity 的 7 个实体。
* 音乐管理
    * 音乐信息实体
    * 用户歌单

### 领域事件
用户注册时，创建默认歌单，里面包含随机的歌曲。
